1.Structure de base d'une page html

Explication des balises clés

!DOCTYPE html

: Cette déclaration est la première ligne de code et indique au navigateur que le document est une page HTML5.

html

: C'est la balise racine qui englobe tout le contenu de la page. L'attribut lang="fr" spécifie la langue du document.

head

: Cette section contient des informations sur le document, qui ne sont pas directement affichées sur la page elle-même.

meta charset="utf-8"

: Spécifie l'encodage des caractères, essentiel pour l'affichage correct des accents et des caractères spéciaux.

title

: Définit le titre qui apparaît dans l'onglet ou la barre de titre du navigateur et dans les résultats de recherche.

body

: Cette section contient tout le contenu visible de la page web, comme le texte, les images, les liens, etc..
